Fila

Operações sobre Pilhas:
init/clear() – inicializa/limpa a pilha;
isEmpty() – verifica se a pilha está vazia;
push(elem) – coloca o elemento no topo da pilha;
pop() – toma o elemento mais alto da pilha;
show() – imprime o conteúdo da fila
showp() – exibe o conteúdo dos ponteiros auxiliares
Uma fila é simplesmente uma lista linear de informações, que é acessada na ordem primeiro a entrar, primeiro a sair, também chamada de FIFO (First In, First Out).
Isto é, o primeiro item a ser colocado na fila é o primeiro item a ser retirado, o segundo colocado é o segundo a ser recuperado e assim por diante.
Essa é a única forma de armazenar e recuperar em uma fila; não é permitido o acesso randômico a nenhum item específico.

                            def init():
                                global fila, spos, rpos, MAX

                                MAX = 5
                                fila = []
                                spos = 0
                                rpos = 0

                                return

Pop()

                              
                                def pop():
                                    global rpos, fila

                                    if rpos == spos:
                                        print("Fila Vazia!!")
                                        return

                                    rpos = rpos + 1
                                    return fila[rpos-1]

push()


                                def push(valor):

                                    global spos, fila
                                    if spos == MAX:
                                        print("Fila cheia!!!")
                                        return

                                    fila.append(valor)
                                    spos = spos+1
                                    return

show()


                                def show():
                                    global spos, rpos, fila
                                    if rpos == spos:
                                        print("Fila vazia!!")
                                        return

                                    for i in range(rpos, spos):
                                        print(fila[i])

                                    return

showp()


                                def showp():
                                    print("rpos = ", rpos," spos = ", spos," MAX = ", MAX)
                                    return
